# Triple Peaks Coffee Shop

This is the second project of the Software Engineering program at TripleTen. It was created using HTML and CSS, based on the design brief.


## Project Description 
The Coffee Shop website is designed to provide users with information about our coffee shop located in the Triple Peaks Library. It features sections for recipes, table reservations, menu items, and general information about the coffee shop. The website is built using HTML and CSS, ensuring a responsive and user-friendly experience.

## Project features
- Semantic HTML5
- Flexbox
- Positioning
- Flat BEM file structure
- A custom form
- CSS animation and transform
